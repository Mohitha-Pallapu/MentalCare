# 🧠 AI Mental Health Risk Analyzer

A mobile-style web application that estimates mental health risk based on lifestyle, stress, and personal factors.  
The app provides an interactive risk score, explainable insights, and personalized recommendations.

## 🌐 Live Demo

🔗 https://mental-health-risk-analyzer.streamlit.app/

---

## ✨ Features

- 📱 Mobile-app style user interface
- 🧠 AI-style risk scoring system
- 📊 Dynamic gauge visualization
- 🎯 Color-coded risk levels (Low / Moderate / High)
- 🔍 Explainable risk factors
- 💡 Personalized recommendations
- 🔒 Privacy-focused (no data stored)
- ⚡ Fast and lightweight

## ⚙️ How It Works

The application calculates a mental health risk score using key indicators such as:

- Sleep duration
- Work / academic pressure
- Study / work hours
- Financial stress
- Family history of mental illness
- Age factor

These inputs are processed using a weighted scoring model to produce:

✔ Risk percentage  
✔ Risk category  
✔ Key contributing factors  
✔ Personalized advice
